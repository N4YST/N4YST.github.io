前言

记录一下JavaWeb的学习过程，试着写个简单的网站

一些基础

网页和网站

  • 网站：是指在网络上根据一定的规则，使用HTML等指作的用于展示特定内容相关的网页合集
  • 网页：就是网站中的一“页”，通常是HTML格式的文件，需要通过浏览器阅读
  • 网页是构成网站的基本元素，它通常由图片、链接、文字、声音、视频等元素组成，通常以htm或html后缀结尾，因此统称为HTML文件

HTML

  • HTML指的是超文本标记语言，全称(Hyper Text Markup Language)
    • 超文本：指超过了文本限制的一种格式，可以加入图片、音频、动画、链接
  • HTML 不是一种编程语言，而是一种标记语言(markup language)
  • 标记语言是一套标记标签(markup tag)
  • HTML 使用标记标签来描述网页

网页的形成

网页是由网页元素构成的，这些元素是利用html标签描述出来的，通过浏览器解析后才有可读性

常用的浏览器有：

  • 谷歌（Chrome）
  • 火狐（Firefox）
  • IE
  • Safari
  • Opera

Web标准

Web标准是由W3C组织（万维网联盟：国际著名的标准化组织）和其他标准化组织指定的一系列标准的集合

  • 构成
标准 说明
结构（Structure） 结构用于对网页元素进行整理和分类，主要学HTML
表现（Presentation） 表现用于设置网页元素的板式、颜色、大小等外观样式、主要指的是CSS
行为（Behavior） 行为指的是网页模型的定义以及交互的编写，主要学的是JavaScript
  • Web标准提出最佳体验方案：结构、样式、行为相分离

静态web

  1. htm、html都是网页的后缀，如果服务器上存在这些东西，我们就可以直接进行读取

  1. 缺点
    • 页面无法动态更新，且所有用户看到的都是同一个页面
      • 伪动态：轮播图、点击特效
      • JS
      • VBScript
    • 无法和数据库交互（数据无法持久化，数据需要定期手动更新）

动态web

  1. 页面会动态展示，给用户展示个性化的界面、根据用户的喜好推送

  1. 缺点：
    • 如果动态资源出错，则需要重新编写后台程序，重新发布
      • 停机维护
  2. 优点：
    • 页面可以动态更新，且所有用户看到的都不是同一个页面
    • 可以和数据库交互（用户注册、商品信息、用户信息）

JavaWeb

Web服务器

服务器端脚本语言

  1. ASP（Active Server Pages）

    • 由微软开发，在国内最早流行
    • 在HTML中嵌入了VB的脚本
    • 在ASP开发中，基本每个页面都有上千行业务代码，极其混乱
    • 维护成本高
    • 主要使用C#
    • 使用IIS

  2. JSP/Servlet（Java Server Pages）

    • 由sun公司开发，主推B/S架构
    • 基于Java语言
    • 可以承载三高
    • 语法类似ASP

  3. PHP（原Personal Home Page，现已正式更名为PHP: Hypertext Preprocessor）

    • 适用于中小型网站开发（国内占70%，WP）
    • 开发速度快、功能强大、跨平台、代码简单
    • 无法承载大访问量（缺少并发）

常见服务器

  1. IIS（Internet Information Services，互联网信息服务器）

    • 由微软开发
    • 使用ASP

  2. Tomcat

  • 是Apache 软件基金会（Apache Software Foundation）的Jakarta 项目中的一个核心项目

  • 由Apache、Sun 和其他一些公司及个人共同开发而成

  • Tomcat 服务器是一个免费的开放源代码的Web 应用服务器，属于轻量级应用服务器，在中小型系统和并发访问用户不是很多的场合下被普遍使用，是开发和调试JSP 程序的首选

  • Apache 为HTML页面服务，而Tomcat 实际上运行JSP 页面和Servlet

Servlet

Servlet简介

  • Servlet 就是 sun 公司开发动态web的一门技术
  • Sun 在这些 API 中提供一个接口叫做 Servlet，如果你想开发一个Servlet程序，只需要：
    1. 编写一个类，实现Servlet接口
    2. 把开发好的Java类部署到Web服务器中
  • 把实现了Servlet接口的Java程序叫做Servlet

Hello Servlet

  1. 创建一个普通的Maven项目，删掉里面的src目录，当做我们学习用的项目，建好的空工程就是Maven主工程

  2. 父工程中的java，子项目可以直接使用，不需要重复添加

  3. Maven环境优化

    1. 修改Web.xml为最新的
    2. 将Maven的结构搭建完整

  4. 编写一个Servlet程序

    1. 编写一个普通类

    2. 实现Servlet接口，直接继承HttpServlet

      public class HelloServlet extends HttpServlet {
    @Override
    protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        //ServletOutputStream outputStream = resp.getOutputStream();
        PrintWriter writer = resp.getWriter();
        writer.print("Hello,Servlet ");
    }

    @Override
    protected void doPost(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        doGet(req, resp);
    }
}

  5. 编写Servlet的映射

    为什么需要映射：我们写的是java程序，但是通过浏览器访问需要链接web服务器，所以需要在web服务中注册我们写的Servlet，还需要给他一个浏览器能够访问的路径 

    <!--注册Servlet-->
   <servlet>
       <servlet-name>hello</servlet-name>
       <servlet-class>top.nayst.servlet.HelloServlet</servlet-class>
   </servlet>
<!--Servlet请求路径-->
   <servlet-mapping>
       <servlet-name>hello</servlet-name>
       <url-pattern>/hello</url-pattern>
   </servlet-mapping>

  6. 配置Tomcat

  7. 启动测试

HTML

学习HTML5基础，主要学习的最常见的没有兼容性的HTML标签

元素

HTML 元素指的是从开始标签（start tag）到结束标签（end tag）的所有代码。

  1. HTML元素语法

    • HTML 元素以开始标签起始
    • HTML 元素以结束标签终止
    • 元素的内容是开始标签与结束标签之间的内容
    • 某些 HTML 元素具有空内容（empty content）
    • 空元素在开始标签中进行关闭（以开始标签的结束而结束）
    • 大多数 HTML 元素可拥有属性

  2. 嵌套的HTML元素

    大多数 HTML 元素可以嵌套（可以包含其他 HTML 元素）。

    HTML 文档由嵌套的 HTML 元素构成。

<html>

<body>
	<p>This is my first paragraph.</p>
</body>

</html>

上面的案例包含三个HTML元素

文档类型声明

<!DOCTYPE> 声明帮助浏览器正确地显示网页。

Web 世界中存在许多不同的文档。只有了解文档的类型，浏览器才能正确地显示文档。

HTML 也有多个不同的版本，只有完全明白页面中使用的确切 HTML 版本，浏览器才能完全正确地显示出 HTML 页面。这就是 <!DOCTYPE> 的用处。

  • 注意：
    • <!DOCTYPE> 声明位于文档的最前面的位置，处于<html>标签之前
    • <!DOCTYPE> 不是 HTML 标签。它为浏览器提供一项信息（声明），即 HTML 是用什么版本编写的。

编码（字符集）

为了正确显示 HTML 页面，Web 浏览器必须知道要使用哪个字符集。

在<meta>标签中指出，在<head>标签内

<meta charset="UTF-8">

charset常用的值有：GB2312、BIG5、GBK和UTF-8，其中UTF-8被称为万国码，包含了几乎所有国家需要用到的字符

基础

每个网页都有一个基本的结构标签（也称为骨架标签），页面内容也是在这些基本标签上书写

标签名 定义 说明
<html></html> HTML标签 页面中最大的标签，称为根标签
<head></head> 文档的头部 在head标签中我们必须要设置title标签
<title></title> 文档的标题 让页面拥有一个属于自己的网页标题
<body></body> 文章的主体 元素包含文档的所有内容，页面内容都是放到body里面的 
<html>
    <head>
        <title>my first page</title>
    </head>
    <body>
        Hello world!
    </body>
</html>

属性

属性总是以名称/值对的形式出现，比如：*name=”value”*。

属性总是在 HTML 元素的开始标签中规定。

表格

表格由 <table> 标签来定义。每个表格均有若干行（由 <tr> 标签定义），每行被分割为若干单元格（由 <td> 标签定义）。字母 td 指表格数据（table data），即数据单元格的内容。数据单元格可以包含文本、图片、列表、段落、表单、水平线、表格等等。

<table border="1">
<tr>
<td>row 1, cell 1</td>
<td>row 1, cell 2</td>
</tr>
<tr>
<td>row 2, cell 1</td>
<td>row 2, cell 2</td>
</tr>
</table>

列表

无序列表

无序列表是一个项目的列表，此列项目使用粗体圆点（典型的小黑圆圈）进行标记。

无序列表始于 <ul> 标签。每个列表项始于 <li> 。

<ul>
<li>Coffee</li>
<li>Milk</li>
</ul>

有序列表

同样，有序列表也是一列项目，列表项目使用数字进行标记。

有序列表始于 <ol> 标签。每个列表项始于 <li> 标签。

<ol>
<li>Coffee</li>
<li>Milk</li>
</ol>

CSS

学习CSS3基础，主要学习最常用的没有兼容性的CSS样式，利用前面学习的标签可以完成传统PC端网页布局

技术介绍

CSS是层叠样式表单，用于（增强）控制网页样式并允许将样式信息与网页内容分离的一种标记性语言。

语法规则

选择器：浏览器根据选择器决定受CSS样式影响的HTML元素（标签）

属性：要改变的样式名，并且每个属性都有一个值。属性和值用冒号分开，并由花括号包围，这样就组成了一个完整的样式声明，例如p{color:blue}

多个声明：如果要定义不止一个声明，则需要用分号将每个声明分开，最后一条可以不加（但尽量在每条末尾加）

注：一般每行只描述一个属性，增加可读性

注释：/*注释*/

CSS与HTML结合方式

  1. 在标签中直接定义样式
  2. 在页面头部定义模板，在body中直接引用
  3. 把css样式写成一个单独的css文件，再通过link标签引入即可复用

JavaScript

介绍

Javascript 语言诞生主要是完成页面的数据验证。因此它运行在客户端，需要运行浏览器来解析执行JavaScript代码

js是Netscape网景公司的产品，最早取名为Livescript，为了吸引更多java程序员。更名为Javascript

Js是弱类型，Java是强类型

特点:

  1. 交互性（它可以做的就是信息的动态交互)
  2. 安全性〈不允许直接访问本地硬盘)
  3. 跨平台性（只要是可以解释Js的浏览器都可以执行，和平台无关)

与HTML代码的结合方式

  1. 只需要在head中或body中，使用script标签来书写JS代码
  2. 使用script标签引入单独的JS代码文件

变量

  1. 变量类型

    • 数值类型：number 字符串类型：string
    • 对象类型：object
    • 布尔类型：boolean
    • 函数类型：function

  2. 特殊值

    • undefined：未定义，所有js变量未赋予初始值的时候，默认值都是undefined
    • null：空值
    • NAN：Not A Number的缩写，即非数值

  3. 定义变量的格式

    • var 变量名；
    • var 变量名=值；

  4. 关系运算

  5. 逻辑运算

    • 且运算

      当表达式全为真时，返回最后一个表达式的值

      当表达式中有一个为假的时候。返回第一个为假的表达式

    • 或运算

      当表达式全为假，返回最后一个表达式的值

      只要有一个表达式为真，就返回第一个为真的表达式的值

    • 且运算和或运算存在短路，当&&或||运算有结果之后，后面的表达式就不再执行

事件

事件就是电脑输入设备与页面进行交互的响应

常见的事件

  1. onload 加载完成事件： 页面加载完成之后，常用语做页面JS代码初始化
  2. onclick 单击事件： 常用语按钮的点击响应操作
  3. onblur 失去焦点事件： 常用于输入框失去焦点后验证其输入内容是否合法
  4. onchange 内容发生改变事件： 常用于下拉列表和输入框内容发生改变后操作
  5. onsubmit 表单提交事件： 常用于表单提交前，验证所有表单项是否合法

事件的注册（绑定）

事件的注册又分为静态注册和动态注册：

事件的注册就是告诉浏览器，当事件响应后要执行哪些操作代码

静态注册事件：通过HTML标签的事件属性直接赋予事件响应后的代码，这种方式我们叫静态注册

动态注册时间：是指先通过js代码得到标签的dom对象，然后通过dom对象.事件名 = function() {} 这种形式赋予事件响应后的代码

​ 动态注册基本步骤：

​ 1、获取标签对象
​ 2、标签对象.事件名 = function() {}

DOM 模型

全称是Document Object Model 文档对象模型

就是把文档中的标签、属性、文本转换成对象来管理

Document对象

1. Document它管理了所有的HTML文档内容
2. Document它是一种树结构的文档，有层级关系
3. 它让我们把所有的标签都对象化
4. 我们可以通过Document访问所有的标签对象

注意：document对象的三个查询方法的优先级，Id属性>Name属性>TagName属性

  1. createElement

    用于创建一个新标签

window.onLoad = function () {
    
	var divObj = document.createElement("div");//创建标签div
    
	divObj.innerHTML = "good";//给新标签赋值
    
	document.body.appendChild( divObj );
}

  1. getElementById

    通过Id属性选择标签

function  onclickFun () {
    
    var usernameObj = document.getElementById("username");//获取id为username的标签

    var usernameText = usernameObj.value;//可以获取其value值

    var usernameSpanObj = document.getElementById("usernameSpan");//可以控制标签内的内容
}

  1. getElementByName

    通过name属性选择标签

function checkAll () {//让所有复选框选中
    
    //getElementsByName()是根据指定的name属性查询返回多个标签对象集合，操作类似数组
    var hobbies = document.getElementsByName("hobby");

    //checked表示复选框的选中状态，true表示选中、false表示不选，可读、可写
    for( var i = 0; i < hobbies.length; i++)
        hobbies[i].checked=true;
}

  1. getElementByTagName

    通过TagName属性选择标签

function checkAll () {
    
    //按照指定标签名来进行查询并返回集合
    var inputs = document.getElementsByTagName("input");
    
    //操作跟数组一样，都是dom对，也是按照HTML页面中从上到下的顺序
    for (var i = 0; i < inputs.length; i++) {
        inputs[i].checked = true;
    }
}

jQuery

介绍

顾名思义，就是JavaScript和查询（Query），它是辅助JavaScript开发的js类库

核心思想是write less，do more

为按钮添加响应函数的步骤：

  1. 使用jQuery查询到标签对象
  2. 使用标签对象.click( function () {} );

jQuery核心函数

$ 是jQuery的核心函数，可以完成jQuery的很多功能

具体作用：

  1. 传入参数为[函数]时：在文档加载完成后执行这个函数
  2. 传入参数为[HTML字符串]时：根据这个字符串创建元素节点对象
  3. 传入参数为[选择器字符串]时：根据这个字符串查找元素节点对象
  4. 传入参数为[DOM对象]时：将DOM对象包装为jQuery对象返回

jQuery对象与DOM对象

  1. jQuery对象的本质就是 DOM对象的数组 + jQuery提供的一系列功能函数
  2. jQuery对象不能使用DOM对象的属性和方法，DOM对象不能使用jQuery对象的属性和方法
  3. jQuery对象和DOM对象可以相互转化

XML

介绍

  1. 什么是XML？

    • XML 指可扩展标记语言（EXtensible Markup Language）。

    • XML 是一种标记语言，很类似 HTML。

    • XML 的设计宗旨是传输数据，而非显示数据。

    • XML 标签没有被预定义。您需要自行定义标签。

    • XML 被设计为具有自我描述性。

  2. XML和HTML的区别？

    • XML 不是 HTML 的替代。
    • XML 和 HTML 为不同的目的而设计。
    • XML 被设计为传输和存储数据，其焦点是数据的内容。
    • HTML 被设计用来显示数据，其焦点是数据的外观。
    • HTML 旨在显示信息，而 XML 旨在传输信息。

  3. 特点

    1. XML是纯文本文件。
    2. XML允许创作者定义自己的标签和自己的文档结构。
    3. XML是独立于软件和硬件的信息传输工具。
